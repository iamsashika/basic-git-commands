## Basic Git Commands

Configure Git

> `git config --global user.name "Sashika Chandrasena"`

> `git config --global user.email "hello@iamsashika.com"`

Crete new local repository

> `git init`

Get status

> `git status`

Add file

> `git add <file name>`

> `git add *`

Add commits

`git commit -m "Commit message"`

`git commit -a`
